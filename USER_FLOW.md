## Price Calculation Example

### Scenario:
- Document type: Passport Translation
- Pages: 5
- User: Regular (not agency)
- Copies: 2
- Pricing settings:
  - First page: 50,000 sum
  - Other pages: 40,000 sum each
  - Copy percentage: 100%

### Calculation:

```

   First page: 50,000

   Base = 50,000 + 160,000 = 210,000 sum

2ï¸âƒ£ Copy Charge:
   Copy charge = Base Ã— (Copy% / 100) Ã— Number of Copies
   Copy charge = 210,000 Ã— 1.0 Ã— 2 = 420,000 sum

3ï¸âƒ£ Total:
   Total = Base + Copy Charge

```
